# 🌍 TRIPYY

TRIPYY is a smart travel exploration web application that helps users discover Points of Interest (POI), along with historical, geographical, and real-time information about any place in the world.

It integrates multiple powerful APIs to provide a seamless and informative travel experience.

# 🚀 Features

🔍 Search Any Location

Find detailed information about cities, landmarks, and regions.

# 📍 Points of Interest (POI)

Discover nearby attractions, monuments, restaurants, and more.

# 🗺️ Interactive Maps

Visualize locations and explore places using map integration.

# 🛣️ Route Navigation

Get routes and directions between locations.

# 📚 Wikipedia Integration

Access historical and detailed descriptions of places.

# 🌦️ Geo Weather Information

Get real-time weather data for any location.

# 🔌 API Integrations

TRIPYY uses multiple APIs to deliver rich data:

- 📍 POI API – Fetch nearby places and attractions

- 🗺️ Map API – Display maps and location visualization

- 🛣️ Route API – Navigation and directions

- 📖 Wikipedia API – Historical and informational content

- 🌦️ GeoWeather API – Weather updates

# 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript, Bootstrap

- Backend (if applicable): Flask 

# 💡 Use Cases

- Exploring new places
